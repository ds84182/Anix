Class ReadStream

ReadStreams are attached to WriteStreams.

ReadStream:listen (callback)
Listens to all event coming to a ReadStream with a callback. The callback is executed in a new thread in the context of the process that owns this object.

Parameters:

  • callback func Callback function that gets executed for every data item received

Returns:

    ReadStream The ReadStream, used for function chaining
ReadStream:onClose (callback)
Allows the ReadStream to handle closure events. The callback is called when the WriteStream is closed and when this ReadStream is explicitly closed.

Parameters:

  • callback func The callback that is called in a new thread when the ReadStream is closed.

Returns:

    ReadStream The ReadStream, used for function chaining

ReadStream:transform (convert)
Creates a new stream that converts each message of this stream to a new value using the convert function.

nil values from convert are not sent. if you want this to be an error, use ReadStream:expand

convert = function(hasData, data, source). hasData = false: source closing

Parameters:

  • convert func Converts one piece of input data into zero or more pieces of output data.

Returns:

    ReadStream The newly created ReadStream
